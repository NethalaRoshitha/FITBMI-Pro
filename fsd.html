About This Project

FitBMI Pro is a web-based health application that calculates Body Mass Index and provides personalized weight recommendations. It allows users to enter height and weight in different measurement systems and receive instant health feedback.

Technologies Used

  • HTML – Structure of the web page
  • CSS – Styling and layout design
  • JavaScript – BMI calculation and dynamic result generation

This project demonstrates how simple frontend technologies can be used to create useful health analysis tools.
